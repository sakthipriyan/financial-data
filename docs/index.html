Available Data Source

Only one source is currently published:

  • SBI Forex Card Rates (USD)
  • Schema: {"header":["date","tt_buy","tt_sell"],"data":[...]}
  • Files are split by year under docs/sbi-fx-card-rates/<year>/USD.json
