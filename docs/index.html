Published datasets for SBI Forex Card USD TT Buy/Sell history and India CPI Inflation index.

SBI Forex Card Rates (USD)

Compact yearly JSON for ITR reference with schema {"header":["date","tt_buy","tt_sell"],"data":[...]}.

  • Latest year: ./sbi-fx-card-rates/2026/USD.json
  • Previous year: ./sbi-fx-card-rates/2025/USD.json
  • Path format: ./sbi-fx-card-rates/<year>/USD.json
  • Source (historical): created from sbi-fx-ratekeeper
  • Source (ongoing): from 2026-03-31 onwards, data is obtained directly from SBI site via scheduled sync

India CPI Inflation Index

Monthly India CPI Combined index (Base 2024=100) from January 2013 onwards, in a single JSON file.

  • Dataset: ./inflation/IN.json
  • Schema: {"metadata":{...},"data":{"YYYYMM": index_value, ...}}
  • Coverage: January 2013 to present
  • Source: eSankhyiki — MoSPI CPI Macro Indicators
